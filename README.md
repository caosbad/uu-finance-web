# uu-finance-web

## Use
- Vue 3
- Vue CLI 4.x (https://cli.vuejs.org)
- Ant Design of Vue (https://2x.antdv.com)

## Project setup
```
npm install
npm install -g @vue/cli
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
